# Intelligent Enterprise Assistant

AI-powered chatbot built using Python, PyTorch, and NLP.

## Features
- HR query support
- IT helpdesk automation
- Natural language understanding
- Deep learning intent detection

## Technologies
- Python
- PyTorch
- NLTK
- Flask

## Run the project

Install dependencies

pip install torch nltk scikit-learn flask

Train the model

python train.py

Run chatbot

python chatbot.py
